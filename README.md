# PortSwigger Web Security Academy Writeups

This repository contains my personal notes and writeups while solving labs from PortSwigger Web Security Academy.

I created this repo to track my learning process, practice web security concepts hands-on, and document the different vulnerabilities I come across during the labs.

The writeups are intentionally kept simple and practical instead of full professional pentest reports. Most of them are written like short handwritten notes showing:
- the vulnerability
- payloads used
- how the exploit worked
- observations during the lab
- screenshots/results

The main goal is to focus on understanding the concepts rather than just collecting solved labs.

## Topics Covered

- SQL Injection (SQLi)
- Cross-Site Scripting (XSS)
- Authentication vulnerabilities
- Access Control vulnerabilities
- CSRF
- SSRF
- and other web security issues
